- [ ] Iespēja objektu novietot uz pareizā laukuma
- [ ] Iespēja rotēt objektu pulksteņrādītāja un pretējā virzienā
- [ ] Iespēja mainīt objekta izmēru pa x asi
- [ ] Iespēja mainīt objekta izmēru pa y asi
